e-DOC is a Computerized Document Management System (CDMS) capable of handling physical, electronic and hybrid documents. In this project, I designed the application layers and the technological stack. This solution was developed in JEE6, EJB3, JPA2, JTA, Jasper Report and JSF 2.0. The Enterprise Content Management (ECM) database stores the metadata and documents are stored in the GFS2 file system. The databases are in RISC partition, in Oracle RAC.
